Duecare

A Gemma 4 implementation built to stop bad actors and reduce human exploitation and abuse.

A multi-stage safety harness for recruiter ads, chat messages, contracts, complaint letters, and bulk case files. Every input flows through a heuristic prescan, a regex‑and‑RAG knowledge base of statutes and conventions, deterministic domain tools, and a Gemma 4 verdict — with each stage written to an audit trail any reviewer can inspect.

  • ⚡Social-media platforms
    moderation queues, CA AB 587 / Texas HB 20 transparency reports
  • 🛡NGOs & hotlines
    Polaris, IJM, ECPAT, HRD Nepal — bulk case triage
  • ⚖Government & regulators
    DOL, BP2MI, MOM, MOHRE, FET — license & statute enforcement
  • 👤Individuals
    workers & families — self-check a suspicious offer or message
  • 🔬Researchers
    auditable harness for benchmarking, fine-tuning, policy work
