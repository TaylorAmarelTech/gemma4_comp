The science / methodology piece judges read. Smoke benchmark (stock
Gemma 4) → Unsloth SFT (LoRA on 21K labeled prompts) → DPO (preference
pairs from worst↔best graded responses) → GGUF export → HF Hub push of
the fine-tuned weights.
