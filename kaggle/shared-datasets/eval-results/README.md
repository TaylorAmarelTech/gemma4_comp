# Duecare — Evaluation Results

This directory becomes a Kaggle Dataset via `kaggle datasets create -p .` or
`kaggle datasets version -p . -m "..."`.

## Contents (populated by `scripts/publish_kaggle.py`)

```
dataset-metadata.json   ← Kaggle metadata (this repo)
reports/                ← generated by Historian on each workflow run
runs.jsonl              ← WorkflowRun dumps, one per line
domains.json            ← frozen snapshot of shipped domain packs
```

## How reviewers reproduce

```bash
pip install duecare-llm[all]==0.1.0
git checkout <git_sha from runs.jsonl>
duecare run rapid_probe --domain trafficking
```

The resulting report should match `reports/<run_id>.md` byte-for-byte
modulo the timestamp prefix.
