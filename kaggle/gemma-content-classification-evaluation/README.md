# Duecare Gemma Content Classification Evaluation (Core Notebook 5 of 6)

The Agency / NGO dashboard. Form-based content submission → structured
JSON classification with risk vectors + threshold-filterable history
queue + pipeline modal. Same toggleable Persona / GREP / RAG / Tools
harness layers as the chat playground.

Built with Google's Gemma 4 (base model:
[google/gemma-4-e4b-it](https://huggingface.co/google/gemma-4-e4b-it)
and other IT variants). Used in accordance with the
[Gemma Terms of Use](https://ai.google.dev/gemma/terms).

## What it does

Single-form classifier UI:

1. Paste content into the textarea (recruitment post, WhatsApp chat,
   contract, employer letter, complaint narrative, etc.).
2. Optionally upload a document image (passport scan, fee receipt,
   complaint form). The Gemma 4 multimodal path inlines the image into
   the prompt.
3. Toggle the four harness layers (Persona / GREP / RAG / Tools).
4. Click **Classify**.

Returns a structured JSON envelope:

```json
{
  "classification": "predatory_recruitment_debt_bondage",
  "classification_label": "Predatory recruitment with debt bondage",
  "confidence": 0.92,
  "overall_risk": 0.91,
  "risk_vectors": [
    {"dimension": "ilo_forced_labor_indicators",
     "magnitude": 0.95, "direction": "high",
     "evidence": "ILO C029 indicators 4 + 7..."},
    {"dimension": "fee_violation",
     "magnitude": 0.88, "direction": "high",
     "evidence": "68% APR violates HK Money Lenders Ord §24..."}
  ],
  "recommended_action": "escalate_to_regulator",
  "ngo_referrals": ["POEA", "BP2MI", "MfMW HK"]
}
```

History queue at the bottom shows all submissions with a
risk-threshold slider (drag to filter by `overall_risk`). Click any
result for the per-response Pipeline modal showing the byte-level
prompt transformation.

## Bundled examples (16)

The classifier ships with 16 example items judges can click to load:

- 10 text-only (recruitment posts, employer letters, narratives)
- 6 with embedded SVG document mockups (Facebook recruitment post,
  WhatsApp chat thread, fee receipt, employer demand letter, NGO
  intake form, police report)

The SVG mockups exercise the Gemma 4 multimodal path end-to-end and
demonstrate that "real, not faked" works on document images, not just
plain text.

## Wheels included (3)

`duecare-llm-core`, `duecare-llm-models`, `duecare-llm-chat`.

The classifier is a single-file FastAPI app shipped inside
`duecare-llm-chat` (`duecare/chat/classifier.py` +
`classifier_static/`). No additional dependencies beyond the chat
playground stack.

## Publishing

```bash
python scripts/push_kaggle_demo.py --kernel content-classifier \
    --enable-gpu false
```

The `--enable-gpu false` flag bypasses Kaggle's 2-GPU concurrent-session
cap; the user toggles GPU ON in the UI before Save & Run All.
