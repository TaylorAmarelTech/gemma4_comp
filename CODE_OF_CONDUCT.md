# Code of Conduct

## Our standard

Duecare exists to help end migrant-worker trafficking and modern
slavery. Our work is read by NGO partners, regulators, researchers,
and (we hope) survivors. We hold our community to that standard.

In project spaces (issues, PRs, discussions, related Kaggle
notebooks, the chat playground) we expect:

- **Respect for survivors.** Real people's experiences inform every
  rule, every doc, every prompt. Treat their stories with care. Use
  composite character names (Maria, Ramesh, Sita) when illustrative,
  labelled as composites.
- **Respect for peers.** Disagreement is fine; demeaning language is
  not.
- **Care with examples.** Sample prompts are intentionally
  adversarial — they imitate trafficker / recruiter / lender voices.
  Don't use them outside their evaluation context. Don't "joke" with
  them.
- **No real PII.** Never paste a real worker's name, passport number,
  phone, address, employer name (unless the employer is in the
  public court record), or case identifier. Use composites.
- **Privacy is non-negotiable.** No telemetry, no usage tracking,
  no "send a small log line back" — even for debugging. If you need
  to reproduce a bug, work from a synthetic example.

## Unacceptable

- Harassment, doxxing, or threats of any kind, including against
  named NGOs, named regulators, or named recruiters whose practices
  the project critiques.
- Posting real victim names, real passport / iqama / id numbers, real
  phone numbers, real bank accounts.
- Submitting code or prompts crafted to deceive deployed Duecare
  instances (i.e. weaponising the project against the people it's
  trying to protect).
- Misrepresenting the project's track record or claims (e.g. "the
  harness has fine-tuned weights deployed in production NGOs" — it
  doesn't, yet).

## Enforcement

Project maintainers have full discretion to remove comments, reject
PRs, lock issues, ban contributors, and report severe abuse to
relevant authorities. Severe abuse here includes:

- Doxxing or threats against survivors, NGO staff, or contributors.
- Use of the project to actively assist trafficking, recruitment fraud,
  or financial crime targeting workers.
- Persistent posting of real PII after warning.

Report concerns to the maintainer at **amarel.taylor.s@gmail.com**.
