Duecare embedded

This page demonstrates the duecare-widget.js single-file embed. It's a complete chat surface — message bubbles, suggested prompts, streaming response — wrapped into one <script> tag.

To embed on your own site (NGO website, partner intake portal, internal tool, etc.):

<div id="duecare-chat" style="height:600px"></div>
<script src="duecare-widget.js"></script>
<script>
  Duecare.mount('#duecare-chat', {
    apiUrl: 'https://your-duecare-deploy.example.com',
    toggles: { grep: true, rag: true, tools: true },
    theme: 'light'
  });
</script>

Configure the API endpoint to point at any deployed Duecare instance — Render, Cloud Run, EKS, your own Kubernetes, anywhere from docs/cloud_deployment.md.

Full embedding guide: docs/embedding_guide.md.
